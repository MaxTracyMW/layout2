GUI Layout Toolbox is a set of MATLAB graphics containers that can be used to create graphical user interfaces that resize gracefully. The containers can be used in combination to produce virtually any user interface layout.

  • Arrange components horizontally, vertically or in grids
  • Specify absolute, relative and minimum component sizes
  • Resize components interactively by dragging dividers
  • Show and hide components using tabs and panels

This toolbox was developed by David Sampson and Ben Tordoff from the Consulting Services group at MathWorks.
